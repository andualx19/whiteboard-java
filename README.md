# Whiteboard Java

A Java application that implements a simple whiteboard with client-server communication over HTTP.

## Technologies
- Java
- Swing
- HTTP Server
- IntelliJ IDEA

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/USERNAME/REPO_NAME.git
2. Open the project in IntelliJ IDEA
3. Run the Main class

## Features
- Whiteboard-style drawing
- Client-server communication
- Graphical user interface built with Swing
